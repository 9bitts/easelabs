# Doctor8 × Ease Labs — Landing Page

Landing page da proposta de parceria estratégica entre **Doctor8** e **Ease Labs Pharma**.

## Visualizar localmente

Abra o arquivo `index.html` no navegador, ou use um servidor local:

```bash
npx serve .
```

## Publicar no GitHub Pages

1. Faça push do repositório para `https://github.com/9bitts/easelabs.git`
2. No GitHub, vá em **Settings → Pages**
3. Em **Source**, selecione **Deploy from a branch**
4. Escolha a branch `main` e a pasta `/ (root)`
5. Salve — o site ficará disponível em `https://9bitts.github.io/easelabs/`

## Estrutura

```
easelabs/
├── index.html    # Landing page completa
├── styles.css    # Estilos (gradientes verdes, glass morphism)
├── script.js     # Animações e interatividade
└── README.md
```
